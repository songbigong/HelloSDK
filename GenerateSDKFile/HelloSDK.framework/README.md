## Installation

HelloSDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'HelloSDK', :git => 'git@gitlab.ehuandian.net:ehd-test-ios/common/HelloSDK.git'
```
